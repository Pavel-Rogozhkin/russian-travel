# Путешествие по России

## О проекте:
Лендинг для путешествий по России, с описанием интересных мест для посещения и ссылками на них.
Можно просмотреть карту, расписание электричек и поездов.

## Стек проекта: 
* HTML
* CSS
* БЭМ структура стилей
* Flexbox
* Grid Layout
* Псевдоэлементы
* Адаптивная вёрстка
